# 🌿 Overview

**EcoRoot** is a full-stack sustainability platform developed using the **MERN Stack**.

The project combines:

🌱 Interactive frontend experiences
⚙️ Full-stack backend architecture
🎮 Mini eco-themed browser games
📱 Responsive user interface

The vision behind EcoRoot is simple:

> Encourage eco-conscious thinking through engaging digital experiences.

# ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/codemasteryash/ecoroot.git
```

### Install Dependencies

```bash
npm install
```

### Start Backend

```bash
npm run server
```

### Start Frontend

```bash
npm start
```
